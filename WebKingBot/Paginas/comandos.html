Comandos de Moderacion



Ban

Banear a un Usuario


k!ban [@user] [razon]



Ban ID

Banear a un Usuario por su ID


k!idban [ID]



Un Ban

Desbanear a un Usuario


k!unban [ID]



Kick

Kickear a un Usuario


k!kick [@user] [razon]



Warn

Warnear a un Usuario


k!warn [@user] [razon]



Check Warns

Ver la cantidad de warns tiene el Usuarios


k!checkwarns [@user]



Un Warn

Restaurar los warns del Usuario


k!unwarn [@user]



Mute

Silencia a un Usuario para que no pueda escribir


k!mute [@user] [razon]



Temp Mute

Silencia a un Usuario, durante determinado tiempo


k!tempmute [@user] [tiempo] [razon]



Un Mute

Quitar silencio, para que pueda volver a escribir el Usuario


k!unmute [@user]

k!servericon



Ping

k!ping

k!clear [100]



Say

k!say [texto]

k!esay [texto]



8Ball

k!8ball [pregunta]
